Certificate Chain Detail —
CA Mode

Internal CA (default): certificates are issued by the built-in two-tier PKI (root → intermediate → per-service leaf). No external dependency.

BYO-CA mode: set YASHIGANI_PKI_CA_MODE=byo to use an external signing endpoint (step-ca, Vault PKI, etc.). Requires YASHIGANI_BYO_SIGNING_ENDPOINT and YASHIGANI_BYO_CA_CERT_PATH. See documentation for full configuration reference.
