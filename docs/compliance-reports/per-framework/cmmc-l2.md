## Coverage note

Controls that require operator-side artefacts (Business Associate Agreements, signed risk-management policies, physical-access logs, training records) are **out of automated scope** and must be evidenced through the customer's own GRC programme. For the full control set see the ACS aggregate output at `acs-scan-2026-05-08-postfix/framework__cmmc_2_0_level_2.json`.

FAILs in this report represent **automated check misses** against code or documentation files in this repository. Per `feedback_no_fake_compliance_docs.md` they are reported as FAIL honestly, not silently downgraded.
