# Security Policy

## Supported Versions

Onbly the latest version is supported, all previous have been deprecated and no longer supported.

| Version | Supported          |
| ------- | ------------------ |
| 0.8.0  | :white_check_mark: |
| 0.7.1   | :x:                |
| < 0.7.0  | :x:                |

## Reporting a Vulnerability

Thank you for helping us keep Yashigani secure.
We appreciate any exploitable vulnerability submission to bugs at agnosticsec.com

Agnostic Security do not pay bounties for bugs, you'll be added to the wall of fame of contributors.
Only vulnerabilities in Yashigany code will be considered, any vulnerabilities in 3rd party components should be reported to them.
